# Fliporia

Fliporia is a browser-based memory card matching game built with HTML, CSS, JavaScript, and Vue.js.

## Features
- Username login
- Easy, medium, and hard difficulty levels
- Theme switching
- Timer and move counter
- Score and high score tracking
- Responsive layout for desktop and mobile

## Technologies Used
- HTML
- CSS
- JavaScript
- Vue.js
